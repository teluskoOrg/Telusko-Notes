# Welcome to Java

Java is a powerful, platform-independent programming language used for building enterprise applications, mobile apps, and web services.

## Why Learn Java?

Java remains one of the most popular programming languages due to its:

- **Platform Independence**: Write once, run anywhere with JVM
- **Object-Oriented**: Clean, modular code structure
- **Rich Ecosystem**: Extensive libraries and frameworks
- **Strong Community**: Vast resources and support
- **Career Opportunities**: High demand in enterprise development

<Callout type="info">
Java powers billions of devices worldwide, from smartphones to enterprise servers.
</Callout>

## Quick Start

<Tabs items={['Windows', 'macOS', 'Linux']}>
  <Tab value="Windows">
    ```bash
    # Download JDK from Oracle or OpenJDK
    # Install and set JAVA_HOME
    java --version
    ```
  </Tab>
  <Tab value="macOS">
    ```bash
    brew install openjdk
    java --version
    ```
  </Tab>
  <Tab value="Linux">
    ```bash
    sudo apt update
    sudo apt install default-jdk
    java --version
    ```
  </Tab>
</Tabs>

## Your First Program

```java title="HelloWorld.java"
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

Compile and run:

```bash
javac HelloWorld.java
java HelloWorld
```

## What's Next?

Explore the following topics to master Java:

- **Fundamentals**: Variables, data types, and operators
- **OOP Concepts**: Classes, objects, inheritance, and polymorphism
- **Collections**: Lists, sets, maps, and streams
- **Advanced Topics**: Multithreading, generics, and lambdas

<Callout type="warn">
Make sure you have JDK 8 or higher installed for modern Java features.
</Callout>
